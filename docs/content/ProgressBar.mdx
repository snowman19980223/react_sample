## Examples

```jsx live
<ProgressBar progress={80} />
```

### Inline

If you'd like to use ProgressBar inline, pass the `inline` boolean prop & **be sure to set a width**.

```jsx live
<>
  <Text mr={3}>5 of 10</Text>
  <ProgressBar progress={50} inline sx={{width: '100px'}} />
</>
```
