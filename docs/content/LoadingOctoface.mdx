---
title: LoadingOctoface
---

The LoadingOctoface component is used as a loading animation.

## Default example

```jsx live
<LoadingOctoface/>
```

## System props

LoadingOctoface components get `COMMON` system props. Read our [System Props](/system-props) doc page for a full list of available props.

## Component props

LoadingOctoface does not get any additional props other than the `COMMON` system props.
