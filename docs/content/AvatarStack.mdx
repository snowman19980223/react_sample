---
title: AvatarStack
---

AvatarStack is used to display more than one Avatar in an inline stack.

AvatarStack
```jsx live
 <AvatarStack>
  <AvatarStack.Item src="https://avatars.githubusercontent.com/primer"/>
  <AvatarStack.Item src="https://avatars.githubusercontent.com/github"/>
  <AvatarStack.Item src="https://avatars.githubusercontent.com/emplums"/>
  <AvatarStack.Item src="https://avatars.githubusercontent.com/jonrohan"/>
  <AvatarStack.Item src="https://avatars.githubusercontent.com/shawnbot"/>
 </AvatarStack>
 ```

## System props

AvatarStack components get `COMMON` system props. Read our [System Props](/system-props) doc page for a full list of available props.

## Component props

AvatarStack components do not get any additional props beyond system props.
