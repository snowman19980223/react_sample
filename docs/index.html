Using
<NavLink>
from react-router

To use UnderlineNav with react-router or react-router-dom, pass

tag={NavLink}
and omit the
selected
prop. This ensures that the NavLink gets
activeClassName='selected'
.

FooTwoSelected
<UnderlineNav>
  <UnderlineNavLink tag={NavLink} to="#foo">Foo</UnderlineNavLink>
  <UnderlineNavLink tag={NavLink} to="#bar">Two</UnderlineNavLink>
  <UnderlineNavLink tag={NavLink} to="/">Selected</UnderlineNavLink>
</UnderlineNav>
